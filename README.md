# ModManager

管理本地 Mod 的桌面工具箱——浏览、启用、禁用、搜索、重命名，
还能一键从 GameBanana 下载安装。

## 功能

- **Mod 管理**：三级分类（类型 → 角色 → Mod），一目了然
- **启用 / 禁用**：右键切换，自动加删 `DISABLED` 前缀，不影响原文件
- **按键编辑**：3DMigoto 配置文件可视化编辑，不用手写 INI
- **预设管理**：保存多套按键配置方案，一键切换不同组合
- **角色搜索**：按元素、武器、性别、体型、地区等标签快速筛选角色
- **一键下载**：浏览器扩展抓取 GameBanana 链接，直接装进对应角色目录
- **搜索过滤**：Mod 管理页支持 AND/OR 多词组合搜索和筛选模式切换


## 安装

> **系统要求**：Windows 10 1809 或更高版本。
>
> **运行环境**：[.NET 10 运行时](https://dotnet.microsoft.com/download/dotnet/10.0) + [Windows App SDK 运行时](https://learn.microsoft.com/windows/apps/windows-app-sdk/downloads)。



## 浏览器扩展

`chrome://extensions` → 开启开发者模式 → 加载已解压的扩展 → 选择 `Extension` 文件夹。

之后在 GameBanana 的 Mod 页面点击"Send to ModManager"即可自动下载。
